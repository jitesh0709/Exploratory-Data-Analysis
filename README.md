# Exploratory Data Analysis on Insurance Data

This repository contains an Exploratory Data Analysis (EDA) project on an insurance-related dataset.  
The main goal of this project is to understand the structure of the data, clean it, and discover meaningful patterns and insights through visualizations and summary statistics.

## Objectives

- Load and inspect the dataset (shape, columns, data types).
- Handle missing values, duplicates, and inconsistent entries.
- Explore distributions of key features using charts and summary statistics.
- Analyze relationships between variables (correlations, group-wise analysis).
- Present important insights and observations derived from the data.

## Tech Stack

- Python
- Jupyter Notebook
- pandas, numpy
- matplotlib, seaborn
